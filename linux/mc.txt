Команды

	не может выделить весь текст в файле

	f3 - открыть

	f5 - копировать

	f8 - удаляет файл или директорию

	f7 - поиск в файле

	shift + left button mouse - выделяет кусок текста

	ctrl + shift + c - копирует текст

	ctrl + shift + v - вставляет текст

	end - в конец документа

	f10 - выход

	ctrl + o - открывает консоль на том месте, где нахожусь и вовращает обратно mc при повторном нажатии

	alt + shift + ? - открываем окно настроек для глобального поиска

	+ - если находимся в директории и нужно выделить все файлы (после нажатия откроется окно в котором нужно указать *)
