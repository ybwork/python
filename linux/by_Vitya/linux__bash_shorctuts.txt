
Bash (сочетания клавиш)


===========
Перемещение
===========
Ctrl + A (Home)             курсор в начало строки
Ctrl + E (End)              курсор в конец строки
Alt + F / Alt + B           назад/вперед на слово


==============
Редактирование
==============
Ctrl + Y                    снова вставить только что удаленный текст
Ctrl + U                    вырезать все до начала строки
Ctrl + K                    вырезать все до конца строки
Ctrl + W                    удаление слова
Alt + T                     поменять слово местами с предыдущим


========
Процессы
========
Ctrl + C        отменить программу
Ctrl + D        Сингал конца файла (окончания ввода)
Ctrl + Z        Перевести в фоновый режим (восстановить - с помощью fg и bg)


==============
История команд
==============
Tab             автозавершение названия файла или команды
↑/↓             листание просмотра команд, введенных ранее (после настройки .inputrc появляется фильтрация - вводишь команду и листаешь только начинающиеся так же)
Сtrl+R          поиск такой команды в истории команд (далее нажимать Ctrl+R - поиск назад)


======
Разное
======
Нажатие Alt+"." (Esc+".") вызывает последний упомянутый объект. Например, если набрать cat /etc/apt/sources.list, а затем rm и нажать Alt+".", то команда будет автоматически завершена: rm /etc/apt/sources.list.
Ctrl + x; Ctrl + e открывает $EDITOR для изменения введённой строки. После сохранения изменений, команда отправляется на исполнение.
Чтобы запустить команду, требующую sudo, набирать ее прямо в редакторе: sudo ...моя_длинная_команда...
