	Это система управления пакетами.

Установка

	Поставляется вместе с python.

Команды

	pip freeze -l - выводит список всех установленных пакетов

	pip freeze > requirements.txt - записывает в файл имена всех установленных пакетов

	pip install -r requirements.txt - читает из файла имена пакетов и устанавливает их
