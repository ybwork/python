	Это система управления пакетами.

Установка

	Поставляется вместе с python.

Команды

	pip freeze -l - выводит список всех установленных пакетов

	pip freeze > requirements.txt - записывает в файл имена всех установленных пакетов

	pip freeze | xargs pip uninstall -y - удаляет все установленные пакеты

	pip install -r requirements.txt - читает из файла имена пакетов и устанавливает их

	pip install -U -r requirements.txt - читает из файла имена пакетов и устанавливает их, обновляя до нужных версий (-U)

	pip install --upgrade package_name - обновляет заданный пакет до последней версии

Нюансы

	Есть проекты, где виртуальное окружение и все его зависимости создаются автоматом. Например выполняется команда python bootstrap.py, которая создает magic-env и ставит в него все зависимости. В некоторых случаях pip может поставить зависимости не из файла requirements.txt, а из глобально установленных пакетов на компе.
