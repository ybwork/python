Продукт

    id продукта равен коду карточки товара из magic.

    Продукт валидный и может появиться на сайте если:

        published = True (таблица products)

        есть цены (таблица prices)

        есть на остатках (таблица remains)

    Описанные выше значения искать в базе korallmicro (postgresql).

Как проверить должен товар находиться на сайте или нет

	select * from public.products where id = 1987;

	select * from public.prices where pid = 1987;

	select * from public.remains where pid = 1987;
