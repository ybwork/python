Что это

	Можно установить наши пакеты от которых зависит проекты в режиме разработки.

Установка

	git clone git@gitlab0.km-union.ru:abakarov_ag/py-env-utils.git

	cd py-env-utils

	sudo python setup.py install

Команды

	py_env_utils --install-packages --packages=all - подтягивает пакеты (рядом с проектом должна появиться директория project_name_packages)

	py_env_utils --apply-frontend --verbose --gulp-mode=gulp-simple

	py_env_utils --checkout-packages --packages=dev --verbose

	py_env_utils --remove-pycs --packages='dev' --verbose

Привязка к pycharm

	Открываем пакет, как проект в новом окне.

	Закрываем пакет.

	В pycharm, где главный проект File -> Open -> Директория с пакетом -> Attach

	Директория с пакетом должена появить ниже директории с проектом
