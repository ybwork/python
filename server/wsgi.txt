Что это

	Это протокол взаимодействия между python программой и web сервером.

	Протокол это стандарт.

	Описан в pep 333.

Требования к приложению

	Должно быть вызываемым объектом (функцией или методом)

	Должна принимать 2 параметра:

		environ - словарь переменных окружения

		start_response - обработчик запроса, вызывается внутри функции, описывается в WSGI-сервере

	Пример:

		def simple_app(environ, start_response):
		    status = '200 OK'
		    response_headers = [('Content-type', 'text/plain')]
		    start_response(status, response_headers)
		    return ['Hello world!']

Запуск приложения

	Для запуска приложения необходим application сервер.

	Application сервер это реализация протокола wsgi.

	Виды application серверов:

		uwsgi (web и application сервер)

		gunicorn (application сервер)

		mod_wsgi

		waitress

WSGI-сервер

	Нужен для запуска приложения.

	Запускает WSGI-приложение один раз при каждом HTTP запросе от клиента.

	Формирует переменные окружения (environ)

	Описываем функцию обработчик запроса (start_response)
