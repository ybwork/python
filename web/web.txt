Что это

	Это система доступа к связанным между собой документам, которые находятся на различных компьютерах подключенных к сети интернет.

    Это систему ещё называют - всемирная паутина (World Wide Web или WWW).

    Всемирную паутину образуют сотни миллионов веб-серверов. 

    Большинство ресурсов всемирной паутины основаны на технологии гипертекста.  

    Большая часть информации в всемирной паутине представляет собой гипертекст.

    Для создания, хранения и отображения гипертекста используют язык HTML (HyperText Markup Language).

    Гипертекстовые документы, размещаемые во всемирной паутине, называются веб-страницами.

    Несколько веб-страниц, объединённых общей темой, дизайном называются веб-сайтом.

    Для определения местонахождения веб-сайта во всемирной паутине используются URL-ы.

    URL - это единый указатель на ресурс, например http://google.com.

    Термин URL потихоньку устаревает, а на его смену приходит термин URI.

    URI - это также единый указатель на ресурс.

История

	1989 - Тим Бернерс-Ли предложил концепцию всемирной паутины.

	1989 - 1991 - Тим Бернерс-Ли создал http протокол, язык разметки HTML и первый в мире веб-сервер httpd.

    1991 - Тим Бернерс-Ли создал первый в мире веб-сайт по адресу http://info.cern.ch

   	1991 - всемирная паутина стала общедоступна в интернете

   	1993 - появился знаменитый веб-браузер NCSA Mosaic.

    1995 - был образован объединение всемирной паутины (W3C).

   	2011 - была принята резолюция ООН, признающая доступ в Интернет базовым правом человека.
